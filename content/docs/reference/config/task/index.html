Configuration affecting tasks definition.

Task - a task is used to configure a container and its context of execution

The following example create a task a train task

  tasks:
    - name: RecommenderTrainRF
      train:
        image: eu.gcr.io/myelin-development/recommender-train-rf:1.0.1
