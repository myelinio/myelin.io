Configuration affecting tasks definition.

Task - a task is used to configure a container and its context of execution

The following example create a task a train task

  tasks:
    - name: RecommenderTrainRF
      train:
        image: eu.gcr.io/myelin-development/recommender-train-rf:1.0.1

Task

Configuration for tasks.

Field Type Description Required
name string

The name of the task. This name should be unique in the axon.

 Yes
triggers Trigger[]

The list of triggers. If empty, by default the axon will execute the task only once at axon creation

 No
conditions Condition[]

List of conditions for task execution.

 No
tasks TaskTask[][]

List of task tasks for task execution.

 Yes
parameters Parameter[]

List of parameters for task execution.

 No
